# [IF2143] Proyek SOLID OOP Python
## Pertemuan 12: Dokumentasi dan Version Control

### Deskripsi Proyek
Proyek ini mendemonstrasikan implementasi prinsip SOLID (SRP, OCP, DIP)
pada sistem registrasi mata kuliah menggunakan abstraksi (interface)
dan dependency injection. Selain itu, proyek ini juga menambahkan
docstring dengan format **Google Style** serta pencatatan event sistem
menggunakan modul `logging`.

### Struktur File
- `registrasi_refactor.py`  
  Kode inti yang sudah direfaktor dan ditambahkan logging.
- `README.md`  
  Dokumen ini.

### Cara Menjalankan
1. Pastikan Python 3.x sudah terinstal.
2. Buka terminal pada folder proyek.
3. Jalankan perintah:

   ```bash
   python "registrasi_refactor.py"
